## Guardrail

The project must not drift into:

```text
OpenKoda installation
Terraform showcase
AWS managed architecture demo
Spring Boot sample app project
simple monitoring dashboard project
```

The project should remain focused on:

```text
VM-based tier separation
operating configuration
failure reproduction
log/metric/command evidence
recovery validation
incident documentation
```
